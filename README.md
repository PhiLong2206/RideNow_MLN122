# RIDE NOW — Từ cạnh tranh đến độc quyền số

## Giới thiệu sản phẩm

Website thuyết trình cuộn dọc dành cho môn MLN122, được thiết kế để thay thế PowerPoint khi trình chiếu trên lớp. Trải nghiệm gồm các chuyển động kể chuyện, sơ đồ minh họa và hệ thống điều hướng theo tiến độ đọc.

## Chủ đề học thuật

- Chương 4: Cạnh tranh và độc quyền trong nền kinh tế thị trường.
- Nội dung lựa chọn: 4.3.1. Biểu hiện mới của độc quyền.
- Trọng tâm: dữ liệu, thuật toán, hiệu ứng mạng và hệ sinh thái số.

## Chạy trên máy cá nhân

```bash
npm install
npm run dev
```

## Build production

```bash
npm run build
```

## Triển khai Vercel

- Framework: Vite
- Build command: `npm run build`
- Output directory: `dist`

## Cấu trúc nội dung thuyết trình

Website đi từ vấn đề mở đầu, cơ sở lý thuyết, bốn biểu hiện mới của độc quyền số, mô phỏng RideNow, liên hệ thực tiễn, giá trị của lý thuyết, giải pháp, kết luận đến nguồn tham khảo.

## Minh bạch AI

AI được dùng để hỗ trợ ý tưởng trải nghiệm, cấu trúc nội dung, thiết kế giao diện và lập trình. Nhóm chịu trách nhiệm đối chiếu, chỉnh sửa và trình bày nội dung học thuật.

## Lưu ý

RideNow, MoveGo và FastTrip là các tình huống giả định phục vụ học tập. Website không kết luận bất kỳ doanh nghiệp có thật nào là độc quyền theo pháp luật.
